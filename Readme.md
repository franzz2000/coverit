CoverIt
=======

Introducción
------------

En ocasiones nos vemos con el problema de tener que compartir una captura de pantalla o video con una persona y no nos interesa que aparezca una zona de la misma por contener información restringida.

Esto hace que sea necesario editar la captura o el video para difuminar esa información.

CoverIt permite cubrir mediante recuadros o pixelados partes de la pantalla para que no puedan ser visibles antes de realizar la captura, lo que hace que esta tarea sea más rápida.

De esta manera es fácil ocultar información sensible en caso de compartir escritorios en videoconferencias o previo a realizar una captura de pantalla.

Uso
---

Una vez ejecutado el programa aparece un recuadro de color azul, que se puede mover y redimensionar.

Pulsando el botón derecho sobre el mismo aparecen diferentes opciones.

1. Nueva ventana: Permite crear una ventana nueva para cubrir otra parte de la pantalla
2. Cerrar ventana: Elimina ventana seleccionada
3. Cambiar color: Una vez seleccionado esta opción, adoptará el color situado debajo del puntero del ratón al hacer click.
4. Intercambia Transparencia: Permite hacer que la ventana sea semitransparente para que poder redimensionar con mayor facilidad.
4. Pixelar: Pixela la zona debajo del recuadro con píxeles de un tamaño definido.
Nota: Los recuadros pixelados no se pueden redimensionar, para poder redimensionarlos hay que seleccionar un color para el recuadro.
5. Cambiar tamaño pixel: Permite cambiar el tamaño de los píxeles del recuadro seleccionado. En caso de no haberse pixelado, se pixelará.
6. Salir: Sale del programa
